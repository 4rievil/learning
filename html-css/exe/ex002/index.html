PARÁGRAFOS E QUEBRAS DE LINHA

Um parágrafo pode ser escrito de qualquer maneira, não importa se, dentro do código, a linha está quebrando dentro das tags p e /p. Dentro das marcações (tags) de parágrafo... você pode fazer o que quiser.


Apenas com a tag <br> usada anteriormente, dentro ou fora das marcações, a linha será quebrada.
